Explanation of Dirichlet Parameters

These numpy arrays define various configurations of Dirichlet parameters across experts and significant players:

  • IID: Each expert has identical Dirichlet parameters: [2, 2, 2, 5].
    This implies a symmetric and identical skill distribution for the significant players across all experts, with 5 units reserved for the rest (non-significant or remaining players).
  • Non-IID with Difference Across Significant Players: All experts use [1, 2, 3, 5] as Dirichlet parameters.
    This introduces asymmetry in the significant players' skill distributions, but the structure is consistent across all experts.
  • Non-IID with Difference Across Experts: All experts share equal relative strengths across significant players (e.g., [x, x, x]), but the total weight differs.
    For example:
    • [1.5, 1.5, 1.5, 6.5]
    • [2, 2, 2, 5]
    • [2.5, 2.5, 2.5, 3.5]
    • [3, 3, 3, 2]
    This simulates experts with the same belief pattern but different confidence or scale.
  • Non-IID with Difference Across Significant Players and Experts: Both the distribution across significant players and the total weight differ for each expert.
    Each expert uses a distinct Dirichlet parameter vector:
    • [1, 2, 3, 5]
    • [1.2, 2.4, 3.6, 3.8]
    • [1.4,2.8,4.2,2.6]
    • [1.6,3.2,4.8,1.4]
    This reflects maximum heterogeneity in expert belief structures across both individual players and total influence.
